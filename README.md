# 信号分析-MatLab简单学习

> 本仓库为上海大学中欧工程技术学院的课程《信号分析》的Matlab实验代码。

使用代码前请注意将`work`路径放到`Matlab`的工作目录下，使用多个自定义函数：

## 自定义函数说明
1. `u(t)`单位阶跃信号生成函数
2. `delta(t)`单位脉冲信号生成函数
3. `conv_m(y1,ny1,y2,ny2)`卷积计算补充函数
   - `y`: 信号
   - `ny`: 信号长度
   - 返回值：卷积结果：[x,nx]
4. `isequal_s(x,y)`
   - `x`: 信号
   - `y`: 信号
   - 返回值：是否相等(布尔值)
5. `save_figure_as_image(figure,filename)`
   - `figure`: 图
   - `filename`: 文件名
6. `clear_all`，清除函数
   - 一键清除所有变量，关闭所有图形窗口，清除命令行窗口的内容
